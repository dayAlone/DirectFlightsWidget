# Direct Flights Widget

React-компонент для отображения прямых рейсов с интерактивным выбором дат и времени.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

## Использование

```jsx
import FlightWidget from './components/FlightWidget'

function App() {
  return <FlightWidget />
}
```
